🌱 Automatic Lawn Mower System (IoT-Based)

This project presents the design and implementation of an IoT-based automatic lawn mower system aimed at improving efficiency, reducing manual labor, and enabling smart lawn maintenance.

The system uses sensors, embedded systems, and mobile application integration to autonomously navigate and cut grass.

🚀 Features

🤖 Autonomous navigation
🚧 Obstacle detection using Ultrasonic Sensor
📱 Mobile control via Bluetooth App
⚡ Powered by LiPo Battery
🌍 Eco-friendly (low emissions)
🎛 Adjustable cutting mechanism
🧠 Working Principle

The system is controlled by an Arduino Uno, which acts as the brain of the robot.

Ultrasonic sensors detect obstacles
L293D Motor Driver controls movement
BLDC motor drives cutting blade
Bluetooth module enables remote control

The robot processes sensor data and adjusts movement in real-time.

🔧 Hardware Components

Arduino UNO
L293D Motor Driver Shield
HC-SR04 Ultrasonic Sensor
BLDC Motor
DC Gear Motors
Servo Motor
HC-05 Bluetooth Module
LiPo Battery (11.1V / 14.8V)
